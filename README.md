# Hardwired Design

This repository contains a curated collection of "vintage" electronic
schematics recreated in KiCad format, alongside original hardware designs
for the Hardwired Design project. It serves as a resource for circuit design,
PCB layouts, and 3D models.

## Project Overview

This project is an open resource that includes reference vintage circuitry
with the designs developed for the Hardwired Design project - a public
synthesizer design and manufacturing project. All schematics and layouts
are maintained using modern EDA tools to ensure they are accessible for
contemporary manufacturing and hobbyist use.

## Setup

To get started with these designs, follow the instructions below.

### 1. Install KiCad

This project is designed for **KiCad v10.0**. Please ensure you have the
correct version installed to avoid compatibility issues with the schema and
layout files.

*   Download: [KiCad Official Website](https://www.kicad.org/download/)

### 2. Clone the Repository

Clone this repository to your local machine using Git:

```bash
git clone https://github.com/mdeeds/hardwired-design.git
```
